<!-- ================  Using Comments ============ -->
{{< note title="Comments">}}
**Single line comment**

```python
#This is a comment
print("Hello, World!")
```

**Inline comment**

```python
print("Hello, World!") #This is a comment
```

**Multi-line comments**

```python
#This is a comment
#written in
#more than just one line
print("Hello, World!")
```

or

```python
"""
This is a comment
written in
more than just one line
"""
print("Hello, World!")
```

{{< /note >}}
